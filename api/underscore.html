Underscore

underscore : object

A lightweight underscore implementation. 包括字符串工具、对象工具、函数工具、语言增强等。设计原则：

  1. 与 Lodash 重合的功能与其保持接口一致， 文档: https://github.com/exports/exports
  2. Lodash 中不包含的部分，如有需要可联系 yangjvn14 (Hi) 文档：本文件中函数注释。

Kind: global namespace

underscore.keysIn(object) ⇒ Array

Creates an array of the own and inherited enumerable property names of object.

Kind: static method of underscore
Returns: Array - Returns the array of property names.

Param Type Description
object Object The object to query.

underscore.forOwn(object, iteratee) ⇒ Object

Iterates over own enumerable string keyed properties of an object and invokes iteratee for each property. The iteratee is invoked with three arguments: (value, key, object). Iteratee functions may exit iteration early by explicitly returning false.

Kind: static method of underscore
Returns: Object - Returs object.

Param Type Description
object Object The object to iterate over.
iteratee function The function invoked per iteration.

underscore.toArray(value) ⇒ Array

Converts value to an array.

Kind: static method of underscore
Returns: Array - Returns the converted array.

Param Type Description
value any The value to convert.

underscore.forEach(collection, iteratee) ⇒ undefined

Iterates over elements of collection and invokes iteratee for each element. The iteratee is invoked with three arguments: (value, index|key, collection).

Kind: static method of underscore
Returns: undefined - Just like Array.prototype.forEach

Param Type Description
collection Array | Object The collection to iterate over.
iteratee function The function invoked per iteration.

underscore.map(collection, iteratee) ⇒ Array

Creates an array of values by running each element in collection thru iteratee. The iteratee is invoked with three arguments: (value, index|key, collection).

Kind: static method of underscore
Returns: Array - Returns the new mapped array.

Param Type Description
collection Array | Object The collection to iterate over.
iteratee function The function invoked per iteration.

underscore.slice(collection, start, end) ⇒ Array

Creates a slice of array from start up to, but not including, end.

Kind: static method of underscore
Returns: Array - Returns the slice of array.

Param Type Description
collection Array The array to slice.
start number The start position.
end number The end position.

underscore.splice(collection) ⇒ Array

This method is based on JavaScript Array.prototype.splice

Kind: static method of underscore
Returns: Array - the spliced result

Param Type Description
collection Collection the collection to be spliced

underscore.split(str) ⇒ Array

This method is based on JavaScript String.prototype.split

Kind: static method of underscore
Returns: Array - Returns the string segments.

Param Type Description
str string the string to be splited.

underscore.findIndex(array, [predicate], [fromIndex]) ⇒ number

Find and return the index of the first element predicate returns truthy for instead of the element itself.

Kind: static method of underscore
Returns: number - Returns the index of the found element, else -1.

Param Type Default Description
array Array The array to inspect.
[predicate] function _.identity The function invoked per iteration.
[fromIndex] number 0 The index to search from.

underscore.format(fmt) ⇒ string

The missing string formatting function for JavaScript.

Kind: static method of underscore
Returns: string - The result string.

Param Type Description
fmt string The format string (can only contain "%s")

Example 

format("foo%sfoo", "bar");   // returns "foobarfoo"

underscore.defaults(object, ...sources) ⇒ Object

Assigns own and inherited enumerable string keyed properties of source objects to the destination object for all destination properties that resolve to undefined. Source objects are applied from left to right. Once a property is set, additional values of the same property are ignored.

Kind: static method of underscore
Returns: Object - Returns object.

Param Type Description
object Object The destination object.
...sources Object The source objects.

underscore.isObject(value) ⇒ boolean

Checks if value is the language type of Object. (e.g. arrays, functions, objects, regexes, new Number(0), and new String(''))

Kind: static method of underscore
Returns: boolean - Returns true if value is an object, else false.

Param Type Description
value any The value to check.

underscore.isString(value) ⇒ boolean

Checks if value is classified as a String primitive or object.

Kind: static method of underscore
Returns: boolean - Returns true if value is a string, else false.

Param Type Description
value any The value to check.

underscore.contains(list, value, fromIndex) ⇒ number

Uses indexOf internally, if list is an Array. Use fromIndex to start your search at a given index. http://underscorejs.org/#contains

Kind: static method of underscore
Returns: number - Returns true if the value is present in the list, false otherwise.

Param Type Description
list Array | string the list of items in which to find
value any the value to find
fromIndex number Optional, default to 0

underscore.isRegExp(value) ⇒ boolean

Checks if value is classified as a RegExp object.

Kind: static method of underscore
Returns: boolean - Returns true if value is a RegExp, else false.

Param Type Description
value any The value to check.

underscore.assign(object, ...source) ⇒ Object

Assigns own enumerable string keyed properties of source objects to the destination object. Source objects are applied from left to right. Subsequent sources overwrite property assignments of previous sources.

Note: This method mutates object and is loosely based on Object.assign.

Kind: static method of underscore
Returns: Object - Returns object.

Param Type Description
object Object The destination object.
...source Object The source objects.

underscore.defaultsDeep(object, ...sources) ⇒ Object

This method is like _.defaults except that it recursively assigns default properties.

Kind: static method of underscore
Returns: Object - Returns object.

Param Type Description
object Object The destination object.
...sources Object The source objects.

underscore.fromPairs(pairs) ⇒ Object

The inverse of _.toPairs; this method returns an object composed from key-value pairs.

Kind: static method of underscore
Returns: Object - Returns the new object.

Param Type Description
pairs Array The key-value pairs.

underscore.isArray(value) ⇒ boolean

Checks if value is classified as an Array object.

Kind: static method of underscore
Returns: boolean - Returns true if value is an array, else false.

Param Type Description
value any The value to check.

underscore.isEmpty(value) ⇒ boolean

Checks if value is an empty object, collection, map, or set. Objects are considered empty if they have no own enumerable string keyed properties.

Kind: static method of underscore
Returns: boolean - Returns true if value is an array, else false.

Param Type Description
value any The value to check.

underscore.negate(predicate) ⇒ function

Creates a function that negates the result of the predicate func. The func predicate is invoked with the this binding and arguments of the created function.

Kind: static method of underscore
Returns: function - Returns the new negated function.

Param Type Description
predicate function The predicate to negate.

underscore.partial(func, ...partials) ⇒ function

Creates a function that invokes func with partials prepended to the arguments it receives. This method is like _.bind except it does not alter the this binding.

Kind: static method of underscore
Returns: function - Returns the new partially applied function.

Param Type Description
func function The function to partially apply arguments to.
...partials any The arguments to be partially applied.

underscore.partialRight(func, ...partials) ⇒ function

This method is like _.partial except that partially applied arguments are appended to the arguments it receives.

Kind: static method of underscore
Returns: function - Returns the new partially applied function.

Param Type Description
func function The function to partially apply arguments to.
...partials any The arguments to be partially applied.

underscore.wrap(value, wrapper) ⇒ function

Creates a function that provides value to wrapper as its first argument. Any additional arguments provided to the function are appended to those provided to the wrapper. The wrapper is invoked with the this binding of the created function.\

Kind: static method of underscore
Returns: function - Returns the new function.

Param Type Description
value * The value to wrap.
wrapper function The wrapper function.

为类型构造器建立继承关系

Kind: static method of underscore

Param Type Description
subClass function 子类构造器
superClass function 父类构造器
